# Portfolio Project

## Deployment
- **Frontend**: [https://portfolio-amber-nine-c7ciranvuy.vercel.app](https://portfolio-amber-nine-c7ciranvuy.vercel.app)
- **Backend**: Currently running locally or on Render (update this if deployed)

## Local Development

### Backend
1. Navigate to `backend` directory
2. Install dependencies: `npm install`
3. Create `.env` file with required variables
4. Start server: `npm run dev`

### Frontend
1. Navigate to `frontend` directory
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`
